# Smart Expense Tracker (MERN Stack)

A full-stack expense tracking web application that allows users to securely manage their income and expenses, monitor financial activity, and analyze spending behavior through an intuitive dashboard.

This project is built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)** and demonstrates complete authentication, API integration, and database operations.



## Features

- User Registration & Login (JWT Authentication)
- Secure Password Encryption
- Add Income & Expense Transactions
- Delete Transactions
- Real-time Balance Calculation
- Transaction History
- REST API Integration
- Persistent Database Storage (MongoDB)
- Responsive UI (React + Vite)



## Tech Stack

### Frontend
- React.js
- Vite
- Axios
- CSS

### Backend
- Node.js
- Express.js
- JWT Authentication
- bcrypt.js

### Database
- MongoDB (MongoDB Atlas)
